RAG_Learning
A from-scratch implementation of a Retrieval-Augmented Generation (RAG) chatbot built to learn how modern AI knowledge assistants actually work end to end. The project ingests a PDF (the PostgreSQL 18.4 documentation, ~1,400 pages) and lets you chat with it in natural language: queries are embedded locally with Ollama (nomic-embed-text), the most relevant chunks are retrieved from a Qdrant vector database running in Docker, and the final answer is generated by Google Gemini (gemini-flash-latest). The frontend is a polished dark-themed Streamlit app that mirrors the look-and-feel of ChatGPT/Claude, with sample queries, live status indicators, and conversation memory. The whole stack runs locally — no API costs for embeddings, and the free Gemini tier is enough for chat — making it a practical sandbox for understanding the moving pieces of a production-grade RAG system: PDF loading, chunking with overlap, embedding, vector storage, similarity search, and prompt-grounded generation.

Stack: LangChain · Ollama (nomic-embed-text) · Google Gemini (gemini-flash-latest) · Qdrant · Streamlit · Docker Compose
